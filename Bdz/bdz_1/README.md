# БДЗ №1



Перед освоением дисциплины "Электроника и импульсная техника" или в самом его начале предлагается сформировать первичное представление об основных понятиях и определениях, затрагиваемых в данном курсе, а также повторить некоторые фрагменты из курсов химии, физики и математического анализа, необходимые для освоения компетенций.

## Задание
Необходимо самостоятельно изучить и дать развёрнутые пояснения во время защиты к каждому из пунктов [`перечня необходимых понятий и определений.`](#перечень-необходимых-понятий-и-определений)

### Примечания
1.	Категорически рекомендуется описание каждого из пунктов перечня сопровождать схемой, графиком, формулой.
2.	Каждый пункт рекомендуется описать в объёме 2-5 предложений (без учёта формул, схем и графиков).
3.	Конспект подготавливать не требуется.
4.	Перечень материалов для подготовки не ограничивается, можно пользоваться любыми источниками (книгу «Искусство схемотехники» Хоровиц, Хилл – использовать не рекомендуется в силу её практи-ориентированного, а не академического характера).

## Порядок защиты
Устный опрос по БДЗ-1 выполняется во время лабораторной работы №1. Без успешной защиты БДЗ-1 студент не допускается к защите лабораторной работы №1.
